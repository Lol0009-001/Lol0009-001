# Bus AI Assistant

一個基於 GPS 定位、地理圍欄（geofencing）、AI 對話、以及巴士班次查詢的個人出行助手原型。

## 核心概念
當使用者每隔一段時間更新位置，或進入某個指定區域時，助手會主動詢問想去的目的地，然後根據當前位置提供：

- 最近可用巴士站
- 下一班巴士到站時間
- 最佳路線與轉車方案
- 建議何時開始步行去車站
- 延誤提醒
- 上車／落車提醒

## 一句話版本
> 一個會因應使用者所在位置，主動詢問目的地，並即時幫忙查巴士班次與提醒上車時間的 AI 出行助手。

## MVP 目標
第一版先聚焦於以下能力：

1. 偵測位置
2. 進入指定區域時觸發提問
3. 讓使用者輸入或選擇目的地
4. 查詢附近巴士站與下一班車
5. 顯示最佳候選方案
6. 發送簡單提醒（出發去車站／巴士即將到站）

## 建議技術架構
- 前端：Flutter
- 後端：FastAPI
- AI：LLM 僅用於自然語言理解與對話
- 交通資料：GTFS / GTFS-Realtime / 當地交通局 API
- 通知：本地通知 + 推播
